# Эпатаж — интернет-магазин экипировки для единоборств

Интернет-магазин спортивной экипировки и формы для единоборств с собственным
производством. Каталог с фильтрами, корзина, оформление заказа с доставкой
СДЭК в пункты выдачи, онлайн-оплата через ЮKassa, личный кабинет, отзывы,
возвраты и админ-панель.

---

## Возможности

**Покупателю**
- Каталог с категориями, фильтрами и поиском по названию
- Карточка товара: галерея по цветам, размерная сетка, наличие, отзывы
- Избранное и корзина (работают и для гостя, и для авторизованного)
- Оформление с выбором пункта выдачи СДЭК на карте (Яндекс.Карты)
- Онлайн-оплата (ЮKassa), бесплатная доставка
- Отслеживание заказа по номеру и email (реальные статусы СДЭК)
- Личный кабинет: заказы, возвраты, профиль
- Письма о статусе заказа (создан / оплачен) с данными пункта выдачи

**Магазину (админка)**
- Товары, категории, остатки по размерам, загрузка фото
- Заказы со статусами, передача в СДЭК
- Модерация отзывов, обработка возвратов
- Счётчики новых заказов / отзывов / возвратов в меню

## Технологии

| Слой | Технология |
|------|-----------|
| Бэкенд | Python 3.12, FastAPI |
| База данных | PostgreSQL 16 (SQLAlchemy 2 async, миграции Alembic) |
| Кэш / очереди | Redis 7 |
| Фоновые задачи | arq (воркер: письма, отмена просроченных заказов) |
| Шаблоны | Jinja2 |
| Платежи | ЮKassa |
| Доставка | СДЭК API v2 |
| Карты | Яндекс.Карты JS API |
| Контейнеризация | Docker, docker compose |

## Быстрый старт (локальная разработка)

Требуется установленный Docker.

```bash
# 1. Скопировать пример переменных окружения и заполнить
cp .env.example .env
# отредактировать .env — как минимум DATABASE_URL, SECRET_KEY, ключи ЮKassa

# 2. Поднять контейнеры (app, worker, db, redis)
docker compose up -d

# 3. Применить миграции
docker compose exec app alembic upgrade head

# 4. Открыть в браузере
# http://localhost:8000
```

**Полезные команды**

```bash
docker compose logs app --tail 50      # логи приложения
docker compose logs worker --tail 50   # логи воркера (письма, cron)
docker compose restart app             # перезапуск после правок в Python
docker compose down                    # остановить всё
```

После правок в шаблонах/CSS перезапуск не нужен — обновите страницу
с очисткой кэша (Ctrl+Shift+R). После правок в Python — `restart app`.

## Переменные окружения

Все настройки — в `.env` (не коммитится в git). Шаблон — в `.env.example`.

Основные группы:
- **Общее:** `PROJECT_NAME`, `ENVIRONMENT` (local/production), `DEBUG`
- **БД:** `DATABASE_URL`, `POSTGRES_*`
- **Безопасность:** `SECRET_KEY`, `ALGORITHM`, `ACCESS_TOKEN_EXPIRE_MINUTES`
- **Redis:** `REDIS_URL`
- **Оплата:** `YOOKASSA_SHOP_ID`, `YOOKASSA_SECRET_KEY`, `RECEIPT_VAT_CODE`
- **Доставка СДЭК:** `CDEK_API_URL`, `CDEK_ACCOUNT`, `CDEK_SECURE_PASSWORD`,
  `CDEK_SENDER_CITY_CODE` и др.
- **Почта:** `EMAILS_ENABLED`, `SMTP_*`

## Миграции базы данных

При изменении моделей (`app/models/`) создайте и примените миграцию:

```bash
docker compose exec app alembic revision --autogenerate -m "описание изменения"
docker compose exec app alembic upgrade head
```
