# Эпатаж — интернет-магазин экипировки для единоборств

Интернет-магазин спортивной экипировки (ММА, муай-тай, хоккей) с каталогом, корзиной,
онлайн-оплатой, личным кабинетом, отзывами и админ-панелью.

Проект построен на **FastAPI** с асинхронным стеком, упакован в Docker
и готов к развёртыванию.

---

## Возможности

**Для покупателя**
- Каталог товаров с категориями, вариантами по размеру и галереей фото
- Живой поиск по каталогу с подсказками (фото, цена, категория)
- Корзина (для гостей и авторизованных пользователей) на Redis
- Оформление заказа с онлайн-оплатой через YooKassa
- Личный кабинет: редактирование профиля, смена пароля, история заказов
- Заказы с разделением на активные и завершённые, отдельной страницей заказа,
  фото товаров и отслеживанием статуса
- Отслеживание заказа по номеру для гостей
- Отзывы о товарах (только для купивших, с модерацией)
- Информационные страницы (доставка, возврат, размерная сетка и др.)

**Для администратора**
- Админ-панель на SQLAdmin с группировкой разделов
- Управление каталогом, заказами, отзывами, страницами
- Загрузка фото товаров с автоматической обработкой (обрезка, сжатие)
- Отдельная страница управления складом с подсветкой низких остатков
- Дашборд аналитики (выручка, заказы, средний чек, конверсия)

**Технические особенности**
- Асинхронная работа с БД (SQLAlchemy 2.0 + asyncpg)
- Атомарное списание остатков с защитой от состояния гонки
- Автоотмена неоплаченных заказов через фоновый воркер (arq)
- Подтверждение оплаты через webhook и синхронизацию статуса
- CSRF-защита всех форм
- Ограничение попыток входа (защита от перебора паролей)
- Структурное логирование бизнес-событий
- Хранение времени в UTC с отображением в МСК

## Технологический стек

| Слой | Технология |
|------|-----------|
| Веб-фреймворк | FastAPI |
| ORM | SQLAlchemy 2.0 (async) |
| База данных | PostgreSQL |
| Кэш / корзина | Redis |
| Миграции | Alembic |
| Шаблоны | Jinja2 (SSR) |
| Аутентификация | JWT в httponly-cookie, bcrypt |
| Платежи | YooKassa |
| Фоновые задачи | arq |
| Админка | SQLAdmin |
| Обработка изображений | Pillow |
| Контейнеризация | Docker + Docker Compose |

## Структура проекта

```
app/
├── core/          # конфигурация, БД, Redis, безопасность, логирование, CSRF
├── models/        # модели SQLAlchemy
├── users/         # пользователи, аутентификация, личный кабинет
├── catalog/       # каталог, товары, поиск
├── cart/          # корзина (Redis)
├── orders/        # заказы, оформление, отслеживание
├── payments/      # интеграция с YooKassa
├── reviews/       # отзывы
├── pages/         # информационные страницы
├── admin/         # админ-панель
├── web/           # главная, фильтры шаблонов
├── templates/     # Jinja2-шаблоны
├── static/        # CSS, JS, загруженные фото
└── worker.py      # фоновый воркер (автоотмена заказов)
```

Каждый домен организован по принципу: `models` → `service` → `router`
(модель данных, бизнес-логика, обработчики запросов).

## Запуск для разработки

### Требования
- Docker и Docker Compose
- Python 3.12+ (для миграций с хоста)

### Шаги

1. Клонировать репозиторий и перейти в папку проекта.

2. Создать файл `.env` на основе `.env.example` и заполнить значения:
   ```bash
   cp .env.example .env
   ```

3. Поднять контейнеры:
   ```bash
   docker compose up -d --build
   ```

4. Применить миграции:
   ```bash
   alembic upgrade head
   ```

5. Наполнить каталог тестовыми данными:
   ```bash
   python3 seed.py
   ```

6. Создать администратора:
   ```bash
   python3 create_admin.py
   ```

Приложение будет доступно на `http://localhost:8000`,
админ-панель — на `http://localhost:8000/admin`.
